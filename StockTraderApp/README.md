# StockTrader - אפליקציית מסחר בשוק ההון

אפליקציית אנדרואיד מלאה לסימולציית מסחר בשוק ההון. נכתבה ב-Java.

## תכונות

- **מסך ראשי** - תצוגת תיק כללית, ניווט מהיר וחיפוש מניות
- **15 מניות מוכנות** - AAPL, MSFT, GOOGL, AMZN, META, TSLA, NVDA, ועוד
- **דף מניה מפורט** - מחיר חי, גרף, סטטיסטיקות יום, פוזיציה אישית
- **קנייה ומכירה** - כפתורי כמות מהירים (10/50/Max, 25%/50%/100%), חישוב חי של עלות/תקבול/רווח/הפסד, אישור דיאלוג
- **תיק השקעות** - רשימת כל הפוזיציות הפתוחות עם רווח/הפסד פתוח
- **רשימת מעקב** - מניות שסומנו במועדפים
- **היסטוריית פעולות** - תיעוד כל קנייה/מכירה עם תאריך, מחיר ורווח/הפסד ממומש
- **ניתוח ביצועים** - תשואה כוללת, ביצועי שיא/שפל, אחוז עסקאות מרוויחות, פילוח התיק בעוגה
- **שוק חי** - מחירים מתעדכנים אוטומטית כל 3 שניות

## הוראות הפעלה ב-Android Studio

1. פתח את Android Studio
2. בחר `File > Open` ובחר את התיקייה `StockTraderApp`
3. המתן ל-Gradle Sync שיוריד את התלויות
4. הרץ על אמולטור או מכשיר אמיתי (API 21+)
5. במסך ההתחברות, הזן כל שם משתמש כדי להיכנס (משחק דמו, אין סיסמה)
6. תקבל $100,000 וירטואליים להתחיל לסחור

## ארכיטקטורה

```
com.example.stocktrader/
├── activities/      → 9 מסכים
├── adapters/        → RecyclerView adapters
├── data/            → DataRepository (singleton) + PriceSimulator
├── models/          → Stock, Holding, Transaction, User, Portfolio
└── utils/           → Formatters (מטבע, אחוזים)
```

## טכנולוגיות

- Java
- AndroidX (AppCompat, Material, RecyclerView, CardView, ConstraintLayout)
- MPAndroidChart (גרף קווי וגרף עוגה)
- Min SDK 21 / Target SDK 34

## הערות

- נתונים נשמרים בזיכרון בלבד (לא ב-SQLite/SharedPreferences). יציאה מהאפליקציה מאפסת את המצב.
- מחירי המניות מסומלים פנימית (Random Walk קל). לא מתחבר ל-API חיצוני.
- כל הכפתורים פעילים ומחוברים.
